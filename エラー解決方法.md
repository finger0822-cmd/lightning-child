# Lightningテーマエラーの解決方法

## エラー内容

```
Fatal error: Call to undefined function lightning_get_template_part()
```

このエラーは、親テーマ（Lightning）の関数が見つからないことを示しています。

## 解決方法

### 方法1: 親テーマの再インストール（推奨）

1. WordPress管理画面にログイン
2. 「外観」→「テーマ」を開く
3. Lightningテーマがインストールされているか確認
4. もしインストールされていない場合：
   - 「新規追加」をクリック
   - 「Lightning」を検索してインストール
5. もし既にインストールされている場合：
   - 一度削除してから再インストール

### 方法2: 親テーマファイルの確認

FTP/SFTPでサーバーに接続し、以下のパスを確認：

```
/wp-content/themes/lightning/
```

以下のファイルが存在するか確認：
- `functions.php` - 必須
- `index.php` - 必須
- `style.css` - 必須

### 方法3: 子テーマのfunctions.phpを更新

子テーマの `functions.php` を更新して、親テーマの `functions.php` を明示的に読み込むようにしました。

変更をデプロイしてください。

## 確認手順

1. 親テーマ（Lightning）が正しくインストールされているか確認
2. 子テーマを有効化する前に、まず親テーマを有効化してみる
3. 親テーマが動作することを確認
4. その後、子テーマを有効化

## 注意事項

- 親テーマが正しく動作していないと、子テーマも動作しません
- 親テーマは必ずインストールされている必要があります
- 親テーマと子テーマの両方が有効化されている必要はありません
  - 子テーマを有効化すると、親テーマは自動的に読み込まれます
