# 親テーマ（Lightning）再インストール手順

## 問題

親テーマ（Lightning）がテーマ一覧から消えています。
子テーマを使用するには、親テーマが必ずインストールされている必要があります。

## 解決方法

### 方法1: WordPress管理画面から再インストール（推奨）

1. **WordPress管理画面にログイン**

2. **「外観」→「テーマ」を開く**

3. **「新規追加」をクリック**

4. **Lightningテーマを検索**
   - 検索バーに「Lightning」と入力
   - または「VK」と入力（LightningはVektor社のテーマ）

5. **「インストール」をクリック**

6. **インストール完了後、親テーマを有効化する必要はありません**
   - 子テーマを有効化すると、親テーマは自動的に読み込まれます

7. **子テーマ（Lightning Child）を有効化**

### 方法2: 手動でアップロード

もしWordPress管理画面からインストールできない場合：

1. **Lightningテーマをダウンロード**
   - https://lightning.vektor-inc.co.jp/
   - または WordPress公式テーマディレクトリから

2. **WordPress管理画面からアップロード**
   - 「外観」→「テーマ」→「新規追加」
   - 「テーマをアップロード」をクリック
   - ZIPファイルを選択してアップロード

### 方法3: FTP/SFTPで手動インストール

1. **LightningテーマのZIPファイルをダウンロード**

2. **ZIPファイルを解凍**

3. **FTP/SFTPでサーバーに接続**

4. **以下のディレクトリにアップロード**
   ```
   /wp-content/themes/lightning/
   ```
   （注意：`lightning-child` ではなく `lightning` ディレクトリ）

5. **WordPress管理画面でテーマを確認**

## 確認事項

### インストール後の確認

1. **テーマ一覧に以下が表示されることを確認**
   - ✅ Lightning（親テーマ）
   - ✅ Lightning Child（子テーマ）

2. **親テーマは有効化しない**
   - 親テーマを有効化する必要はありません
   - 子テーマ（Lightning Child）のみを有効化してください

3. **子テーマを有効化**
   - 「Lightning Child」テーマを見つけて「有効化」をクリック

## 重要なポイント

- **親テーマは必ずインストールされている必要があります**
- **親テーマを有効化する必要はありません**
- **子テーマを有効化すると、親テーマは自動的に読み込まれます**

## トラブルシューティング

### 親テーマが表示されない場合

1. **サーバーのファイルシステムを確認**
   - `/wp-content/themes/lightning/` ディレクトリが存在するか確認
   - `functions.php` ファイルが存在するか確認

2. **ファイルのパーミッションを確認**
   - ディレクトリ：755
   - ファイル：644

3. **エラーログを確認**
   - WordPressのデバッグログを確認
