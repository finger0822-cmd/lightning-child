# デプロイ成功後の確認事項

## ✅ デプロイが完了しました！

GitHub Actionsを通じて、WordPressサーバーに以下のファイルがアップロードされました：
- `functions.php`
- `page-ouchi-support.php`
- `screenshot.png`
- `style.css`

## 次のステップ

### 1. WordPress管理画面でテーマを有効化

1. WordPress管理画面にログイン
2. 「外観」→「テーマ」を開く
3. 「BusinessPress Child」テーマを見つけて「有効化」をクリック
   （既に有効化されている場合はそのままでOK）

### 2. 固定ページを作成

「おうち安心サポート」専用ページを作成します：

1. WordPress管理画面で「固定ページ」→「新規追加」
2. ページタイトルを入力（例：「おうち安心サポート」）
3. 右側の「ページ属性」で：
   - 「テンプレート」を「おうち安心サポート」に選択
4. 「公開」をクリック

### 3. ページの確認

作成した固定ページのURLにアクセスして、以下の内容が正しく表示されるか確認してください：

- ✅ ヒーローセクション（タイトル、リード文、ボタン）
- ✅ こんなご家庭におすすめです
- ✅ サービス内容（3つのカード）
- ✅ 料金プラン
- ✅ ご利用の流れ
- ✅ よくあるご質問
- ✅ 対応エリア
- ✅ お問い合わせ・申込み

### 4. 今後の更新方法

**今後は、コードを変更してGitHubにプッシュするだけで自動的にデプロイされます！**

1. ローカルでファイルを編集
2. `git add .`
3. `git commit -m "変更内容"`
4. `git push`

これだけで、GitHub Actionsが自動的にWordPressサーバーにデプロイします。

## デプロイの確認方法

- GitHubの「Actions」タブで、デプロイの実行状況を確認できます
- 緑色のチェックマークが表示されれば、デプロイ成功です
- デプロイログを確認することで、アップロードされたファイルの一覧も見られます

## トラブルシューティング

### ページが表示されない場合

1. テーマが有効化されているか確認
2. 固定ページのテンプレートが正しく選択されているか確認
3. ブラウザのキャッシュをクリア

### スタイルが反映されない場合

1. WordPressのキャッシュをクリア（キャッシュプラグインを使用している場合）
2. ブラウザのキャッシュをクリア
3. ブラウザの開発者ツールでコンソールエラーを確認
