# 親テーマエラーの解決方法

## エラー内容

```
Fatal error: Call to undefined function lightning_get_template_part()
```

このエラーは、親テーマ（Lightning）の `functions.php` が正しく読み込まれていないか、ファイルが不完全であることを示しています。

## 原因

- 親テーマ（Lightning）の `functions.php` が読み込まれていない
- 親テーマのファイルが不完全または破損している
- 親テーマのインストールが不完全

## 解決方法

### 方法1: 親テーマを完全に再インストール（推奨）

1. **WordPress管理画面にログイン**

2. **現在のテーマを確認**
   - 一時的に別のテーマ（Twenty Twenty-Threeなど）を有効化して、サイトを表示可能な状態にする

3. **親テーマ（Lightning）を削除**
   - 「外観」→「テーマ」を開く
   - Lightningテーマが見つかれば削除
   - 見つからない場合は、FTP/SFTPで `/wp-content/themes/lightning/` ディレクトリを削除

4. **親テーマを再インストール**
   - 「外観」→「テーマ」→「新規追加」
   - 「Lightning」を検索
   - 「インストール」をクリック
   - インストールが完了するまで待つ

5. **必要なファイルの確認**
   - `functions.php` が存在するか
   - `index.php` が存在するか
   - `style.css` が存在するか

6. **子テーマを再び有効化**

### 方法2: FTP/SFTPでファイルを確認・修正

1. **FTP/SFTPでサーバーに接続**

2. **親テーマディレクトリを確認**
   ```
   /wp-content/themes/lightning/
   ```

3. **必要なファイルが存在するか確認**
   - `functions.php` ← **これが重要**
   - `index.php`
   - `style.css`
   - その他のテンプレートファイル

4. **`functions.php` が存在しない、またはファイルサイズが0の場合**
   - Lightningテーマの公式サイトから最新版をダウンロード
   - `functions.php` を正しい内容で上書き

### 方法3: Lightningテーマの最新版をダウンロード

1. **Lightning公式サイトにアクセス**
   - https://lightning.vektor-inc.co.jp/
   - または https://wordpress.org/themes/lightning/

2. **最新版のZIPファイルをダウンロード**

3. **ZIPファイルを解凍**

4. **FTP/SFTPで `functions.php` を確認**
   - `functions.php` ファイルが含まれているか確認
   - ファイルサイズが0でないか確認

5. **WordPress管理画面から再インストール**
   - または、FTPでファイルを直接アップロード

## 緊急対応（一時的な回避策）

サイトが完全に表示できない場合：

1. **別のテーマを一時的に有効化**
   - 「外観」→「テーマ」
   - デフォルトテーマ（Twenty Twenty-Threeなど）を有効化

2. **サイトが表示できるようになったら、親テーマを再インストール**

## 確認事項

親テーマが正しくインストールされている場合、以下のファイルが存在する必要があります：

```
/wp-content/themes/lightning/
├── functions.php      ← これが必須
├── index.php
├── style.css
├── header.php
├── footer.php
└── その他のテンプレートファイル
```

特に `functions.php` が存在し、正しい内容が含まれていることが重要です。

## 重要なポイント

- **親テーマが正しく動作しないと、子テーマも動作しません**
- **親テーマの `functions.php` が読み込まれないと、このエラーが発生します**
- **親テーマを完全に再インストールするのが最も確実な解決方法です**
