# おうち安心サポート - スタッフ用チェックリスト

## 訪問前のチェックリスト

### 訪問前日まで
- [ ] 訪問日時を確認
- [ ] 訪問先の住所・連絡先を確認
- [ ] 前回の訪問記録を確認・読み返す
- [ ] 前回からの継続事項・持ち越し事項を確認
- [ ] 必要な工具・道具を準備
- [ ] 消耗品（電球など）を準備（事前連絡があった場合）
- [ ] 訪問チェックリストを印刷・準備

### 訪問当日・出発前
- [ ] 訪問先の情報を再確認
- [ ] 訪問チェックリストを持参
- [ ] 名刺・身分証明書を持参
- [ ] 必要な工具一式を確認
- [ ] 電話・連絡手段を確認
- [ ] 体調管理（マスク・手指消毒など）
- [ ] 出発時間を確認（時間に余裕を持って）

## 訪問後のチェックリスト

### 訪問直後

- [ ] 訪問記録を整理
- [ ] 作業内容を記録
- [ ] 気になる点があれば記録
- [ ] 次回訪問時の確認事項を記録

### 事務所への報告

- [ ] 訪問記録を事務所に提出
- [ ] 重要な事項があれば口頭でも報告
- [ ] ご家族への連絡が必要な場合は事務所に依頼
- [ ] 追加作業・追加料金が必要な場合は事務所に報告

---

## 緊急時・問題発生時の対応

### 緊急時の対応

- [ ] ご本人の安全を最優先に
- [ ] すぐに事務所に連絡
- [ ] 緊急連絡先に連絡（必要な場合）
- [ ] 記録を残す

### 問題発生時の対応

- [ ] 無理に作業を進めない
- [ ] 事務所に相談・報告
- [ ] 専門業者への依頼を提案
- [ ] ご本人・ご家族に説明

---

## 記録のポイント

### 必ず記録すべきこと

1. **作業内容**
   - 実施した作業の詳細
   - 作業時間
   - 使用した部品・消耗品

2. **確認事項**
   - 家電の動作状況
   - 気になる点
   - 次回確認したいこと

3. **ご要望**
   - 次回訪問時の作業予定
   - 追加で必要な作業

4. **ご家族への報告内容**
   - 特に伝えるべき事項
   - ご家族への確認事項

### 記録の形式

- 具体的に、わかりやすく
- 日付・時間を記録
- 次回訪問時に役立つ情報を記録

---

## コミュニケーションのポイント

### ご本人との接し方

- [ ] ゆっくりと、ご本人のペースに合わせる
- [ ] 明るく元気に、安心感を与える
- [ ] 会話を大切に、信頼関係を構築
- [ ] 無理に急かさない

### ご家族との連携

- [ ] 訪問内容をわかりやすく説明
- [ ] 気になる点があれば報告

## 安全・衛生管理

### 安全対策

- [ ] 無理な作業は行わない
- [ ] 高所作業・危険作業は避ける
- [ ] 専門知識が必要な作業は事務所に相談

### 衛生管理

- [ ] マスクの着用
- [ ] 手指の消毒
- [ ] 工具・道具の清潔保持

## 60分を超える場合の対応

- [ ] 作業開始前に時間の見積もり
- [ ] 60分を超えそうな場合は事前にご相談
- [ ] 追加料金が必要な場合は事前に説明
- [ ] 次回訪問に持ち越すことを提案
