## 📋 サービス基本情報

### サービス名
おうち安心サポート

### 提供事業者
株式会社鈴木冷熱

### 基本料金
月額 10,000円（税込）

### サービス内容
- 月2回・各60分の定期訪問
- ちょこっと作業サポート（60分以内の軽作業）
- 設備の簡易点検
- 訪問内容の報告

### サービスコンセプト
「高齢世帯の"小さな不安"を、月2回・60分の定期訪問でやさしく解消する。」

## 🔧 サービス内容の詳細

### 対応可能な作業（60分以内の軽作業）

- 電球交換・電池交換
- リモコン設定
- 家電の動作チェック
- ブレーカーの簡易確認
- タップ・配線の簡易整理
- 小家具の軽移動
- 網戸のはめ直し
- カーテンの付け直し
- フィルター清掃
- 換気扇の簡易清掃
- 扇風機・ストーブの出し入れ
- 外観の簡易雪害チェック
- 生活ヒアリング
- 訪問内容の簡易報告

### 追加料金が必要な作業

- 60分を超える作業
- 技術的な作業（エアコン内部洗浄、ガス補充など）
- 専門工事（エアコン設置、給湯器交換など）

### 対応できない作業

- 屋根の雪下ろし
- 下水・排水の詰まり
- リフォーム工事
- 医療・介護行為
- 金銭・契約の立ち会い
- 危険作業全般
