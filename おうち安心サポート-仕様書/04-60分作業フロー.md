# おうち安心サポート - 60分作業フロー

## 1回の訪問での標準的な流れ

---

## 📋 訪問前の準備（事前確認）

### 訪問3日前まで
- [ ] 訪問日時を確認
- [ ] 前回の訪問記録を確認
- [ ] ご家庭からの連絡事項を確認
- [ ] 必要な道具・部品を準備

### 訪問当日の出発前
- [ ] 訪問先の情報を確認
- [ ] 必要な工具を準備
- [ ] 消耗品（電球など）を準備（必要な場合）
- [ ] 訪問チェックリストを持参

---

## 🏠 訪問時のフロー（60分以内）

### 到着・挨拶（5分）

- [ ] 到着時間の確認
- [ ] ご挨拶・体調確認
- [ ] 訪問の目的を簡潔に確認

**ポイント**：安心感を与える挨拶、明るく元気に

---

### 生活ヒアリング（5〜10分）

- [ ] 前回訪問以降の状況確認
- [ ] お困りごとの有無を確認
- [ ] 季節に応じた確認事項
  - 夏：エアコン・扇風機の動作
  - 冬：ストーブ・ヒーターの動作
  - 季節の変わり目：網戸・カーテンなど

**ポイント**：ゆっくりと、ご本人のペースに合わせて

---

### 軽作業対応（35〜45分）

#### 優先順位1：安全に関する確認
- [ ] ブレーカーの確認
- [ ] 漏電チェック（簡易）
- [ ] ガス器具の確認（目視）

#### 優先順位2：ご要望の作業
- [ ] 電球交換・電池交換
- [ ] リモコン設定
- [ ] 家電の動作チェック
- [ ] フィルター清掃
- [ ] その他ご要望の軽作業

#### 優先順位3：季節に応じた作業
- [ ] 網戸・カーテンの取り付け
- [ ] 扇風機・ストーブの出し入れ
- [ ] 季節家電の動作確認

**ポイント**：
- 1つの作業が長引く場合は、複数の軽作業を組み合わせ
- 60分を超えそうな場合は、事前にご相談
- 安全を最優先に

---

### 作業完了・報告（5分）

- [ ] 実施した作業内容を説明
- [ ] 次回訪問時の確認事項を記録
- [ ] 追加で必要な作業があれば記録
- [ ] ご家族への報告内容を確認

**ポイント**：わかりやすく、簡潔に説明

---

### 退室・次回予約（5分）

- [ ] 次回訪問日の確認
- [ ] 訪問記録の作成
- [ ] ご挨拶・退出

---

```

│  ┌───────────────────────────────┐ │

│  └───────────────────────────────┘ │
├─────────────────────────────────────┤

├─────────────────────────────────────┤
```

---

## ⚠️ 60分を超えそうな場合の対応

### 事前にご相談
- 複数の作業がある場合、優先順位を確認
- 次回訪問に持ち越すことを提案
- 追加料金が必要な場合は事前に説明

### 記録・報告
- 次回訪問時の作業内容として記録
- ご家族にも報告

---

## 📝 訪問記録のポイント

### 記録すべき内容
1. **実施した作業**
   - 作業内容
   - 作業時間
   - 使用した部品・消耗品

2. **確認事項**
   - 家電の動作状況
   - 気になる点
   - 次回確認したいこと

3. **ご要望**
   - 次回訪問時の作業予定
   - 追加で必要な作業

4. **ご家族への報告内容**
   - 特に伝えるべき事項
   - ご家族への確認事項

---

## 💡 効率的な作業のコツ

### 複数作業を組み合わせる
- 同じ場所での作業をまとめる
- 順路を考えて移動時間を短縮

### 事前準備を徹底
- 必要な工具・部品を準備
- 訪問記録で事前に作業内容を把握

### コミュニケーションを大切に
- ご本人のペースに合わせる
- 無理に急かさない
- 会話しながら作業することで、信頼関係を構築

---

## 🔄 次回訪問への引き継ぎ

### 次回訪問時に確認すること
- [ ] 前回の作業の状況確認
- [ ] 持ち越した作業の実施
- [ ] 新たなご要望の確認

---

## 📞 緊急時・疑問点がある場合

- 不明な点があれば、すぐに事務所に連絡
- 安全に関する作業は、無理をせず専門業者を推奨
- ご本人の安全を最優先に

---

**1回の訪問で、安心と信頼を届ける。**
**60分という時間を、最大限に活用して、ご家庭の安心をお届けします。**
