# 緊急対応：ファイルが反映されない問題

## 問題
- デバッグ情報が表示されない
- 料金が9,800円のまま
- 新しいファイルが読み込まれていない

## 考えられる原因

### 1. デプロイ先パスが間違っている
- GitHub Secretsの `FTP_REMOTE_DIR` が正しいパスになっているか確認
- 正しいパス: `public_html/suzukireinetsu.com/wp-content/themes/lightning-child/`
- または: `/public_html/suzukireinetsu.com/wp-content/themes/lightning-child/`

### 2. 別の場所にファイルがある
- 親テーマ (`lightning`) にも `page-ouchi-support.php` がある可能性
- 別の子テーマディレクトリにファイルがある可能性

### 3. WordPressが別のテンプレートを読み込んでいる
- 固定ページの設定で、正しいテンプレートが選択されているか確認

### 4. デプロイが実際には実行されていない
- GitHub Actionsのログを確認して、ファイルがアップロードされているか確認

## 確認手順

### ステップ1: サーバー上のファイルを直接確認
FTPクライアントで以下を確認：

```
/public_html/suzukireinetsu.com/wp-content/themes/lightning-child/page-ouchi-support.php
```

**確認ポイント：**
- ファイルが存在するか
- ファイルの更新日時が最新か（数分以内）
- 10行目に `<!-- DEBUG: ファイルバージョン 2.0` が含まれているか

### ステップ2: 親テーマを確認
```
/public_html/suzukireinetsu.com/wp-content/themes/lightning/page-ouchi-support.php
```
このファイルが存在する場合、WordPressがこちらを読み込んでいる可能性があります。

### ステップ3: WordPressの設定を確認
1. WordPress管理画面にログイン
2. 固定ページ → 該当ページ（ID: 491）を編集
3. 「ページ属性」→「テンプレート」を確認
4. 「おうち安心サポート」が選択されているか確認

### ステップ4: ファイルを手動でアップロード
デプロイが機能していない場合、手動でアップロード：

1. `C:\Users\taker\lightning-child\page-ouchi-support.php` を開く
2. FTPクライアントでサーバーに接続
3. `/public_html/suzukireinetsu.com/wp-content/themes/lightning-child/` にアップロード
4. 既存ファイルを上書き

### ステップ5: テンプレートの読み込み順序を確認
WordPressは以下の順序でテンプレートを探します：
1. 子テーマの `page-ouchi-support.php`
2. 親テーマの `page-ouchi-support.php`
3. 子テーマの `page.php`
4. 親テーマの `page.php`

親テーマに同名ファイルがある場合、子テーマのファイルが優先されます。

## 解決方法

### 方法1: ファイル名を変更して確実に識別
```php
// page-ouchi-support.php の先頭に追加
<?php
/**
 * Template Name: おうち安心サポート
 * Version: 2.0 - 子テーマファイル
 */
```

### 方法2: 一時的にファイル名を変更
`page-ouchi-support-v2.php` などに変更して、テンプレート名も変更：
```php
Template Name: おうち安心サポート V2
```

### 方法3: 直接ファイルを確認して上書き
FTPでサーバー上のファイルをダウンロードして、内容を確認。
10,000円になっていない場合は、手動でアップロード。

## 確認用コマンド（サーバー上で実行可能な場合）
```bash
# ファイルの存在確認
ls -la /home/c0344603/public_html/suzukireinetsu.com/wp-content/themes/lightning-child/page-ouchi-support.php

# ファイルの内容確認（最初の20行）
head -20 /home/c0344603/public_html/suzukireinetsu.com/wp-content/themes/lightning-child/page-ouchi-support.php

# ファイルの更新日時確認
stat /home/c0344603/public_html/suzukireinetsu.com/wp-content/themes/lightning-child/page-ouchi-support.php
```
