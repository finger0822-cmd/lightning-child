# テーマ整理手順

## 現在の状況

- ✅ 親テーマ（Lightning）はインストールされている
- ❓ 親テーマがテーマ一覧に表示されない
- ⚠️ 子テーマが2つ表示されている

## 考えられる原因

### 1. 古い子テーマディレクトリが残っている

サーバー上に以下の両方が存在している可能性があります：
- `/wp-content/themes/businesspress-child/`（古い）
- `/wp-content/themes/lightning-child/`（新しい）

### 2. 親テーマが正しく認識されていない

親テーマのファイルが破損しているか、`style.css`のヘッダーに問題がある可能性があります。

## 解決手順

### ステップ1: サーバー上のディレクトリを確認

FTP/SFTPでサーバーに接続し、以下のディレクトリを確認してください：

```
/wp-content/themes/
├── lightning/              ← 親テーマ（必須）
├── lightning-child/        ← 新しい子テーマ（使用する）
└── businesspress-child/    ← 古い子テーマ（削除可能）
```

### ステップ2: 不要な子テーマを削除

古い `businesspress-child` ディレクトリが存在する場合：

1. FTP/SFTPでサーバーに接続
2. `/wp-content/themes/businesspress-child/` ディレクトリを削除
   - または、WordPress管理画面から古いテーマを削除

### ステップ3: 親テーマの確認

1. `/wp-content/themes/lightning/` ディレクトリが存在するか確認
2. 以下のファイルが存在するか確認：
   - `style.css`
   - `functions.php`
   - `index.php`

### ステップ4: 子テーマの確認

`/wp-content/themes/lightning-child/style.css` を確認：

```css
/*
 Theme Name: Lightning Child
 Template: lightning    ← これが重要
 ...
*/
```

`Template:` の値が `lightning` であることを確認してください。

### ステップ5: WordPress管理画面で確認

1. 「外観」→「テーマ」を開く
2. 表示されている2つの子テーマを確認
3. 正しい子テーマ（Lightning Child）を有効化
4. 不要な子テーマを削除

## 重要なポイント

- **親テーマは有効化する必要がありません**
- **子テーマ（Lightning Child）のみを有効化してください**
- **親テーマがインストールされていれば、表示されなくても動作します**

## トラブルシューティング

### 親テーマが表示されない場合

これは正常な場合もあります。親テーマは有効化する必要がないため、WordPressが表示を省略することがあります。

重要なのは：
- ✅ `/wp-content/themes/lightning/` ディレクトリが存在する
- ✅ 子テーマの `Template:` が `lightning` に設定されている
- ✅ 子テーマが正常に動作する

この3つが揃っていれば問題ありません。
