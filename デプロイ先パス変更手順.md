# デプロイ先パスの変更手順

## 現在のデプロイ先

```
/public_html/suzukireinetsu.com/wp-content/themes/lightning-child/
```

## 変更方法

デプロイ先パスを変更するには、**GitHub Secrets**を更新する必要があります。

### ステップ1: GitHub Secretsを更新

1. GitHubリポジトリにアクセス：
   https://github.com/finger0822-cmd/businesspress-child/settings/secrets/actions

2. `FTP_REMOTE_DIR` のSecretを探して、編集（または削除して再作成）

3. 新しいパスを設定：
   - Name: `FTP_REMOTE_DIR`
   - Secret: `【新しいパスをここに入力】`

### ステップ2: 新しいパスを教えてください

以下の情報をお知らせください：
- 新しいデプロイ先パス（例: `/public_html/suzukireinetsu.com/wp-content/themes/lightning-child/`）

この情報があれば、ドキュメントも更新します。
